## Conway's Game Of Life

This is a simple game I made using [SFML](https://www.sfml-dev.org/) under about 3 hours! This was a little challenge I set up for myself because I was bored.
I will propably add some more features to make it a little cleaner (like a gen count and a resizeable grid), but that's it overall.

I will provide the .exe file, the controls for now are: 


Arrow Key to go: up/down/left/right
Mouse Buttons  :
- Left Click -> The selected Cell comes to life
- Right Click -> The selected Cell gets unalived

Have fun :)

(PS: if you click outside the grid the program may crash xD)
